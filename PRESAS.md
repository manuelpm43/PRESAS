# PROYECTO PRESAS

## Objetivo

Desarrollar una aplicación web para la gestión técnica de presas utilizando la API-IDEE.

---

# Arquitectura

La aplicación estará formada por cinco bloques principales:

- Header
- Panel de capas
- Mapa
- Panel de información
- Footer
